# Expense Tracker (CLI)

A command-line application for tracking personal expenses, built in Python as a learning project.

## Features

- Add expenses with amount, category, and description
- View all recorded expenses
- Delete an expense
- View a summary of total spending and spending by category
- Data persists between runs using a local JSON file

## How to run

1. Clone this repository
2. Make sure Python 3 is installed
3. Run the program:
   \`\`\`
   python main.py
   \`\`\`
